Pixi.Application

  • PIXI.Application class to create a new PIXI Application
  • This class automatically creates the renderer, ticker and root container

Options

The optional renderer parameters

autoStart

  • Type: boolean
  • Default:true
  • Optional: yes
  • Description: Automatically starts the rendering after the construction
  • Note: Setting this parameter to false does NOT stop the shared ticker even if you set option.sharedTicket to true in case that is already started. Stop it by your own

width

  • Type: number
  • Default: 800
  • Description: The width of the renderers view

height

  • Type: number
  • Default: 600
  • Description: The height of the renderers view

view

  • Type: HTMLCanvasElement
  • Default: 600
  • Description: The canvas to use as a view option

transparent

  • Type: boolean
  • Default: false
  • Description: If the render view is transparent

autoDensity

  • Type: boolean
  • Default: false
  • Description: If the render view is transparent

antialias

  • Type: boolean
  • Default: false
  • Description: Sets antialias
