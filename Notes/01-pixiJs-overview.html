PixiJS Overview

  • Pixi is an extremely fast 2D sprite rendering engine that helps you to display, animate and manage interactive graphics so that it is easy for you to make games and applciation using JavaScript and other HTML5 technologies
  • It has uncluttered API and includes many useful features
  • Supports texture atlases
  • Provides streamline system for animating sprites (interactive images)
  • It gives you a complete scene graph so you can create hierachies of nested sprites (sprites inside sprites)
  • Lets you attach mouse and touch events directly to sprites
  • Pixi lets you get out of your way so that you can use as much or as little of it ad you want
  • Adapts it to your personal coding style
  • Integrate it seamlessy with other useful frameworks
  • Pixi's API is actually a refinement of a well-worn and battled-tested API pioneered by Macromedia/Adobe Flash
  • Pixi's API is not a game engine which allows for expressive freedom to make anything you like and wrap your own custome game engine around it.
  • Pixi isnt just for games, you can use these same techniques to create any interactive media applications
