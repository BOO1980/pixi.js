Renderer: AutoResize and resize

  • To change the size of the canvas, use the renderers resize method and supply an new width and height
  • To make sure the canvas is resized to match the resolution, set autoResize to true
  • If you want to make the canvas fill the entire window, you can apply this CSS styling and resize the renderer to the size of the browser window
