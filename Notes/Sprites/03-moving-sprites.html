Moving Sprites

  • To move sprites you create a looping function using PIXIs ticker
  • The loop is called a game loop
  • Any code you put inside a game loop will update 60 times per second
  • The delta value represents the amount of fractional lag between frames.
  • You can make the animation independent of the frame rate
  • To move right it is a positive number
  • to move left is a negative number
