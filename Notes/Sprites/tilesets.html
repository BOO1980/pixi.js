Tile set

  • As a game designer you usually be making your sprites using tilesets - aka spritesheets
  • Pixi has convieniant built in ways to help you get a single sprite from a image tileset
  • A tile set is a single image files that contains sub-images
  • Storing and accessing all you game graphics on a tileset is a very processor and memory efficient way to work with graphics
  • You can capture a sub-image from a tileset by defining a rectangular area thats the same size and position as the sub-image you want to extract
