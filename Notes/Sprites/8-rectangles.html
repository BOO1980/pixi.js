Rectangles

  • Pixi has a built in Rectangle object(PIXI.Rectangle) that is a general-purpose object from defining rectangular shapes
  • The rectangle takes four arguments
  • The first two arguments define the rectangles x and y position
  • The last two define the rectangles width and height
  • The rectangle object is a data object
  • PIXI textures have a useful property called frame that can be set to any Rectangle objects
  • The frame crops the texture to the dimension of the Rectangle
